# kollektivtransport--Application--for-Bus-Group-23
Vi vurderer for tiden å utvikle en applikasjon for kollektivtransport, med fokus på bussruter mellom byene Halden, Sarpsborg og Fredrikstad.
Vi har studert behovene til hver persona individuelt ved å presentere hypotetiske scenarier og daglige utfordringer.
Vi har bygget applikasjonen med en moderne og skalerbar arkitektur som kombinerer React og TypeScript på frontend med Node.js på backend.
I denne repositoryen kan dere se alle komponentene som vi har brukt for å lage prototypen, med mulighet for å prøve den ved å laste ned mappen.
Prototypen er for nå kun for brukeren, men i fremtiden skal vi utvide prosjektet med et admin-dashboard.
Prototypen inneholder:
1- Login page
2- Home page
3- Search page
4- PurchaseSummaryPage

For å teste prototypen:
1- Bruk VS Code
2- I VS Code, åpne terminalen fra "public transport"-mappen
3- Kjør npm install eller npm i
4- For å rendere, kjør npm run dev
5- For testing, kjør npm test
6- For å teste hver enkelt komponent, kjør npm test "komponentnavn"
